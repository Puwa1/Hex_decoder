🛰️ GPS Decoder Pro (Ultimate CTk)
โปรแกรมถอดรหัสข้อมูล GPS (JT808 Protocol) ระดับวิศวกรและผู้พัฒนา (Developer) พัฒนาด้วย Python และ UI ล้ำสมัยจาก customtkinter รองรับการถอดรหัสข้อมูลแบบ Real-time, การประมวลผลไฟล์ขนาดใหญ่ (Batch Processing), เครื่องมือคำนวณ Checksum อัจฉริยะ, และระบบปรับแต่งโปรโตคอลในตัว

✨ ฟีเจอร์เด่น (Key Features)
🌓 Dynamic Theme Engine: อินเทอร์เฟซทันสมัย รองรับ Dark Mode, Light Mode และปรับตาม System อัตโนมัติ

🔍 Manual Decode: วางโค้ด Hex (0200, 0704 หรืออื่นๆ) ลงไปแล้วกดถอดรหัสได้ทันที โปรแกรมจะคัดแยกและวิเคราะห์ให้อัตโนมัติ

📂 File Processing: รองรับการโหลดไฟล์ข้อมูลดิบ (.csv, .txt) มาประมวลผลทีละหลายๆ บรรทัด พร้อมระบบแบ่งหน้า (Pagination) ไม่กินแรม

👁️ Advanced Column Manager: คลิกขวาที่ตารางเพื่อ ซ่อน/โชว์ คอลัมน์ได้อย่างอิสระ ระบบจะจำค่าคอลัมน์ไว้ให้ (ซิงค์กันทุกหน้าต่าง)

🛠️ Smart Checksum Tool:

Synchronized Highlight: ลากคลุม Hex โชว์ ASCII, ลากคลุม ASCII โชว์ Hex สัมพันธ์กันแบบ 2:1 แบบเรียลไทม์

Auto-Verification: หั่น 7E และค้นหา Checksum 2 ตัวท้ายในแพ็กเกจให้อัตโนมัติ เพื่อนำมาเปรียบเทียบ (XOR) ว่าข้อมูลถูกต้องหรือไม่

รองรับ Undo/Redo (Ctrl+Z, Ctrl+Y)

⚙️ Config Editor: แก้ไขโครงสร้างการถอดรหัส (StartByte, Length, DataType) ได้โดยตรงจากหน้าต่างโปรแกรม พร้อมระบบ Import/Export Backup

💾 Export Report: บันทึกผลลัพธ์การถอดรหัสออกเป็นไฟล์ Excel (.xlsx) หรือ CSV พร้อมจัดรูปแบบคอลัมน์ที่เป็นตัวเลข (เช่น IMEI) ให้เป็น Text อัตโนมัติ ป้องกันตัวเลขเพี้ยน

🛠️ ความต้องการของระบบ (Prerequisites)
หากต้องการรันโปรแกรมผ่าน Source Code ตรวจสอบให้แน่ใจว่าได้ติดตั้งไลบรารีเหล่านี้แล้ว:

Bash
pip install pandas customtkinter openpyxl
🚀 การติดตั้งและใช้งาน (Installation & Usage)
นำไฟล์ master_mapping_config.csv (ไฟล์กฎการถอดรหัส) วางไว้ในโฟลเดอร์เดียวกับโปรแกรม

รันโปรแกรมด้วยคำสั่ง:

Bash
python main.py
เลือกแท็บการทำงานที่ต้องการด้านบนของหน้าต่างโปรแกรม

📖 แท็บการทำงาน (Main Tabs)
🔍 Manual Decode: สำหรับทดสอบการถอดรหัสข้อมูล Hex แบบบรรทัดต่อบรรทัด หรือวางเป็นชุดข้อมูล

📂 File Process: สำหรับการนำเข้าไฟล์ CSV ที่มีคอลัมน์ข้อมูลดิบ เพื่อแปลงข้อมูลจำนวนมากเป็นตารางที่อ่านง่าย

🛠️ Checksum Tool: เครื่องมือตรวจสอบความสมบูรณ์ของแพ็กเกจข้อมูล (BCC/CRC-16) และแปลข้อมูล Hex เป็นข้อความ (ASCII)

⚙️ Settings: สำหรับปรับแต่งไฟล์ master_mapping_config.csv แบบ Inline ไม่ต้องเปิดโปรแกรม Excel แยก

📦 การสร้างไฟล์ Executable (.exe)
หากต้องการแปลงโปรแกรมเป็นไฟล์ .exe เพื่อนำไปใช้งานบนเครื่องอื่นโดยไม่ต้องติดตั้ง Python สามารถทำได้ผ่าน PyInstaller:

Bash
pip install pyinstaller
pyinstaller --noconsole --onefile main.py
ข้อควรระวัง: หลังจากสร้าง .exe เสร็จสิ้น (อยู่ในโฟลเดอร์ dist) ต้องก๊อปปี้ไฟล์ master_mapping_config.csv ไปวางไว้ข้างๆ ไฟล์ .exe ด้วยเสมอ เพื่อให้โปรแกรมสามารถโหลดกฎการถอดรหัสได้
